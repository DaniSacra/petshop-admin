# PET SHOP ADMIN

An administrative App for manageing a Pet Shop. Built with Ruby on Rails and Ajax.

## Ruby version

Ruby 
- v 2.5.1

Rails
- v 5.2.0

PostgreSQL
- v 2.0

Clone or fork

    $ bundle install

    $ rails db:create db:migrate

    $ rails s
